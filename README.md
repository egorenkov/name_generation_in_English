# Генерации имён на английском языке

Работа представляет собой систему генерации имён на английском языке с использованием рекуррентной нейронной сети (RNN). RNN — это популярный вид нейронных сетей, используемых в обработке естественного языка (NLP). Особенность таких сетей — наличие «памяти», которая учитывает предшествующую информацию.

Цель проекта — разработать модель, способную генерировать реалистичные и осмысленные имена на английском языке. Для этого модель будет обучаться на большом наборе данных имён, чтобы научиться выявлять закономерности и особенности их структуры.

## Основные компоненты

* Входные данные (x_t) — на каждом временном шаге t модель получает на вход вектор, который может представлять, например, букву или слог имени.
* Скрытое состояние (s_t) — это «память» сети, которая хранит информацию о предыдущих входных данных. Скрытое состояние зависит от предыдущих состояний и текущего входа.
* Выход (o_t) — на каждом шаге модель генерирует вектор вероятностей, который соответствует возможным следующим символам или слогам в имени.
---

### Принцип работы

Модель обучается на последовательности входных данных, предсказывая вероятность каждого следующего элемента. При обучении мы подаём на вход следующему слою предыдущий выход, чтобы модель могла учитывать контекст и генерировать более осмысленные результаты.

Для вычисления скрытого состояния используется функция f, которая обычно является нелинейной:

$s_t = f(Ux_t + Ws_{t-1})$

Первое скрытое состояние $s_{-1}$ инициализируется нулевым вектором.

Выход на шаге t определяется как:

$o_t = softmax(Vs_t)$

где softmax — функция, которая преобразует вектор вероятностей в распределение вероятностей.

Применение

Разработанная модель может быть использована для генерации имён для персонажей в играх, создания имён для виртуальных персонажей в приложениях, а также для тестирования систем обработки естественного языка и других задач.
